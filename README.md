
# newsrack

Generate a "newsrack" of `.mobi` periodicals for your ereader.

Uses [calibre](https://calibre-ebook.com/), [GitHub Actions](.github/workflows/build.yml) and hosted on [GitHub Pages](https://pages.github.com/).
